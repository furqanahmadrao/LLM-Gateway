Security policy
===============

If you believe you have found a security vulnerability in LLM Gateway, please do not open a public issue.

Preferred contact
-----------------
Email: furqanrao091@gmail.com
GitHub: https://github.com/furqanahmadrao

If you are unable to reach the maintainers by email, open a private GitHub security advisory.

Guidance for reporters
----------------------
- Provide a clear description and scope of the issue.
- Include steps to reproduce and any relevant logs, stack traces, or PoC code.
- Provide the version/commit you tested against.

Maintainer response
-------------------
We will acknowledge receipt within 72 hours and provide an estimated timeline for a fix.

After a fix is available, maintainers will coordinate disclosure and release a patched version.
